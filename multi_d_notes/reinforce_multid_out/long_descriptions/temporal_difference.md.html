Overview

Specialized for prediction problems.

Conventional prediction: error between predicted and actual

Temporal Difference: error (difference) between temporally successive predictions

Better than supervised by an actual RL signal because it is incremental.

TD(λ)

The parameter λ introduces a recency bias that makes the method weight more recent samples more highly.

Formally, for
0≤λ≤10 \le \lambda \le 10≤λ≤1, we have

Δwt=α(Pt+1−Pt)∑k=1tλt−k∇wPk\Delta w_t = \alpha (P_{t+1}-P_t) \sum_{k=1}^t \lambda^{t-k}\nabla_w P_kΔwt​=α(Pt+1​−Pt​)k=1∑t​λt−k∇w​Pk​

Note that if λ=1\lambda = 1λ=1 then we have the uniform sampling distribution, where all diffs in the future are weighted equally,
and if λ=0\lambda = 0λ=0 then we have the singleton distribution where only the next diff matters.
